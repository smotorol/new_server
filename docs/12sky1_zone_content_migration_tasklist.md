## 부록 C) 로컬 경로 제공 방법

### C-1. 가장 간단한 방법
1. `12sky1` 폴더(또는 압축 해제본)를 작업 컨테이너에 두고,
2. 절대경로를 그대로 메시지에 적어주면 됨.

예시:
- `/workspace/12sky1`
- `/workspace/imports/12sky1-main`
- `/workspace/uploads/S07_TS_ZONE`

### C-2. 내가 바로 작업 가능한 요청 예시
```text
다음 경로를 분석해줘: /workspace/12sky1/S07_TS_ZONE
- 파일 인벤토리(csv)
- ID 매핑표(md)
- new_server 반영 TODO를 파일 단위로 생성
```

### C-3. 경로 확인 방법 (사용자 측)
- Linux/macOS: `pwd`, `realpath <폴더명>`
- Windows(PowerShell): `Resolve-Path .\12sky1`
- 압축 업로드 후에는 "압축을 /workspace/uploads/12sky1-main 에 풀었어"처럼 **풀린 경로**를 함께 전달

### C-4. 내가 이해하기 좋은 전달 포맷
```text
SOURCE_ROOT=/workspace/12sky1
TARGET=/workspace/12sky1/S07_TS_ZONE
APPLY_TO=/workspace/new_server
OUTPUT=docs/12sky1_analysis
```
